# erp_mobile_app

An open-source Flutter mobile application designed for ERPNext integration, providing a comprehensive solution for project and sales tracking on mobile devices.

## Getting Started

This is an open-source Flutter mobile application designed for ERPNext integration, enabling real-time project and sales tracking.

### Key Features

- Real-time project monitoring
- Sales tracking and management
- ERPNext integration
- Mobile-optimized interface
